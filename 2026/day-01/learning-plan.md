# Understanding of DevOps & Cloud?
# DevOps
- Dev (Development) : The team is responsible for writing code, designing features, and fixing bugs.
Dev builds the product (focus on change).
- Ops (Operations) : The team is responsible for servers, networks, security, scaling and keeping the application running 24/7.
Ops run the product (focus on stability).
- DevOps : Devops is just not a tool; It is a culture and method. It combines Devlopement and Operations into one smooth Team.
- The DveOps infinity loop : Continuous Cycle of Software Delivery :- Plan, Code, Build, Test, Release, Deploy, Operate, Monitor.  

# Cloud
- Cloud : Cloud means storing and accesing data or services over the internet instead of your local computer/laptop.
- On-demand: Access whenever you need.
- Remote: Runs on servers elsewhere, not your device.
- Scalable: Easily increase or decrease resources.
- Pay-as-you-go: You pay only for what you use.
- Examples: Google Drive, AWS, Microsoft Azure.
